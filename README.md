# Nail Spa Inventory Tracker

A beginner-friendly React inventory app for a nail spa. It stores data in `localStorage`, so items stay saved in the browser even after refresh.

## Features

- Add new inventory items
- Edit existing items
- Delete items
- Quick `+1` / `-1` quantity buttons
- Low stock and out-of-stock status badges
- Search by item name, location, or category
- Category filter
- Persistent storage with `localStorage`

## Tech Stack

- React
- Vite
- Plain CSS
- Browser `localStorage`

## Run locally

```bash
npm install
npm run dev
```
